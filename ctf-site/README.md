# NeonCTF (Static) — HTML/CSS/JS

A beginner-friendly CTF challenge platform with:
- 3 difficulty levels (Easy/Medium/Hard)
- 10 tasks per level (30 total)
- Hints, flag submissions, points
- LocalStorage-based sign up/sign in with username + phone + password reset
- Leaderboard
- Downloadable completion certificates (client-side text files)
- Master Challenge unlocked after completing all 3 levels

## Run
Since this is static, open:
- `ctf-site/index.html`

in a browser.

> Note: Because it uses `localStorage`, your progress and accounts are stored only in that browser/device.
